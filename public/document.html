Body Fat Predictor Documentation

Introduction

This document provides an overview of how the Body Fat Predictor project works, including the architecture, the machine learning model, and the tools used in its development.

Project Setup

Before you start, ensure you have the following dependencies installed:

  • Node.js
  • Python 3.x
  • Joblib (for saving and loading the model)
  • Scikit-learn (for machine learning)

To install the dependencies, use the following commands:

npm install pip install -r requirements.txt

Machine Learning Model

The project uses a Linear Regression model to predict body fat percentage based on physical measurements.

The model is trained using the following features:

  • Age: Your current age (years).
  • Weight: Your current weight (kg or lbs).
  • Height: Your height (cm or inches).
  • Neck Circumference: Measurement around the neck (cm or inches).
  • Abdomen Circumference: Measurement around the abdomen (cm or inches).
  • Other body measurements such as chest, hip, thigh, etc.

API Routes

The Node.js server exposes two main routes:

  • GET / - Serves the main page
  • GET /predict - Loads the prediction form
  • POST /submit - Handles predictions
